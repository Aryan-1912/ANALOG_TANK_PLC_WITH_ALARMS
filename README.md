Analog Tank Level Control with HH/LL Alarms
Overview
PLC ladder logic program for tank level monitoring and control using analog input values, with High-High (HH) and Low-Low (LL) alarm notifications.
Project Details
ParameterValuePlatformAllen-Bradley SLC 500 / MicroLogixSoftwareRSLogix 500FileAnalogTank_ALARMS_NOTIFICATIONS.RSS
Features

Analog tank level input processing
High-High (HH) alarm for overflow protection
Low-Low (LL) alarm for empty/dry-run protection
Alarm notification outputs

Alarm Setpoints
AlarmDescriptionHHHigh-High level - triggers when tank is near overflowLLLow-Low level - triggers when tank is near empty
Requirements

RSLogix 500 software to open and edit the program
Compatible Allen-Bradley PLC (SLC 500 or MicroLogix series)
Analog level transmitter (4-20mA)
